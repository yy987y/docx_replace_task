## 三、优先修改建议

建议优先处理顺序：

1. 先处理 `39` 次的“空白/仅单一环节型”
- 这类最显眼，也最容易被人工审看出模板问题。
- 建议补出明确的活动名、目标、准备、过程、观察回应等栏位。

2. 再处理 `22` 次的小团体活动主模板
- 这是当前复用最多的标准模板。
- 建议分拆为至少以下几种版本轮换：
  - `小团体活动 -> 目标 -> 准备 -> 过程 -> 观察与回应`
  - `小团体活动 -> 目标 -> 材料 -> 过程 -> 延伸/观察`
  - `小团体活动 -> 经验准备 -> 活动过程 -> 指导要点 -> 观察回应`
  - `小团体活动 -> 活动目标 -> 活动准备 -> 分环节实施 -> 评价`

3. 处理圆圈活动模板
- 目前主要集中在 `圆圈活动 -> 目标 -> 准备 -> 观察与回应 -> 环节`
- 可以增加“材料探索”“同伴互动”“教师支持”“延伸玩法”等不同结构。

4. 顺手修正同周蓝字重复
- 第五周两处 `yoyo：“好！”`
- 虽不属于跨周问题，但建议也改掉，避免人工复核时被顺手挑出。

- 第 1 条：`通过`
  - 未发现跨周“二级标签 + 蓝色正文内容”完全一致。

- 第 2 条：`不通过`

  - 其中最严重的两类分别达到 `39` 次和 `22` 次。
